Installation

Requirements

To install StreamSite you will only need a few things, nothing too complicated. We have provided you with suggestions on the best/cheapest hosts and domain registrars, to get your site online in the best way possible (You are obviously not required to use our recommendations).

  • Hosting
  • Domain
  • Text editor
  • Twitch account
  • Twitter account
  • For giveaways, a Gleam.io account

Hosting

You will need your own web hosting to be able to upload StreamSite to. We have a few suggestions of hosts we think you should choose that offer a good price and service.

Our recommended hosts.
  • Hostgator - First month of hosting for $0.01
  • BlueHost - $3.95 /month (Includes free domain)
  • iPage - $10.95 /month (Free domain, unlimited disk space) Currently on offer at $1.99/m

Uploading

You will need to upload the contents of the /upload/ folder. There are a few methods you could use to do this, either with an FTP client or via your host's of choices online control panel.

Your host will have a guide and be able to help you upload to the site if you get stuck doing this.

Domain

If your host comes with a free domain then it will be automatically set up for you, however if it doesn't then you will need to register your own domain.

The only domain registrar we recommend is NameCheap, there are others but Namecheap is the only one we have experience with and feel confident recommending. They also provide .stream domains for under $1

To set up your domain, you will need to add your newly purchased domain to your site (Doing so should provide you with 2 Nameservers), thne you will need to change your domain's DNS settings via the Namecheap control panel to the ns servers provided by your host. (Should ressemble ns1.host.com & ns2.host.com)

Customization

Settings

To customize the site to your liking the first thing you will want to do add your own details into the first group of settings.

//Site Settings
var settings = {
"logo" : "img/logo.png", //Your logo image, you can type text too
"twitchUsername" : "twitch", //Your Twitch username
"numberOfVideos" : 4, //Number of broadcasts/highlights to show on page
"giveAwayMessage" : "We are doing a giveaway..", //Message to show when the goal has been reached
"giveawayUrl" : "https://gleam.io/YLJoq/my-giveaway" //Gleam.io giveaway URL, refer to documentation to see how to get this
}

The comments help you understand what needs to be changed, make sure you only edit the parts after the ":"

Sponsor

In this version of StreamSite, we only have added the facility to have one sponsor on the main part of the site, so we suggest you add your main one (Or referal code like in the example)

We are working on having room for multiple sponsors/referal codes, however at this time it destroys the design so we prefer to just work with one for now.

//SPONSOR CONFIG
var sponsor = {
  "main" : {
    "img" : "img/g2a.png", //Image of your main sponsor
    "url" : "http://www.g2a.com", //Website address of your sponsor/affiliate link
    "text" : "
Use coupon #coupon to get 3% off all games
" //Text that appears under the sponsor image
  }
}

Colors

From this part of the customisation page you can change the colors of the site, the comments next to the colors should explain what they are for.

You can put the colors in either RGB (rgb(24, 24, 24)), RGBA (rgba(0, 0, 0, 1)) or hex(#0000) format.

To choose you colors you can use a site like, w3Schools color picker.

RGBA colors are just RGB with an opacity value added on the end, for example : rgba(24, 24, 24, 0.4) is dark gray with 40% opacity.

In future versions we are looking to add the option to add an image as backgrounds for certain elements.

// COLORS
var colors = {
  "header" : "#000", //The background color of the header (Where the logo and stream info is)
  "socialBar" : "rgba(139, 101, 181, 1)", // The color of the bar under the header, where the social media icons are
  "page" : "rgb(24, 24, 24)", //The background color of the page
  "box" : "rgba(0, 0, 0, 1)", //The background color of the content boxes
  "title" : "rgba(139, 101, 181, 1)", //Background color of the title sections on top of the boxes
  "hover" : "#7bbee8" //The color certain elements change to on hover (Social media links for example)
}

Titles

To change the titles of the different sections, change the text below to what you want it to be.

//TITLE CONFIG
var titles = {
  "giveaway" : "Giveaway", //Text for the title of the goal section
  "sponsor" : "Main sponsor", //Title for hte sponsor section
  "video" : "Latest VODs", //Title for the video section
  "tweets" : "Latest Tweets/ReTweets", //Title for the tweets section
  "bio" : "About me" //Title for the Bio section
}

Social media icons

To change a social media icon in the bar underneath the header, edit the url, icon values to what you want them to be.

To get the icon names, go to FontAwesome, click on an icon and copy and paste the "fa-icon-name" into the settings.

Do not change the "id" value otherwise it will not show up.

If you do not put a value into the "url" the icon will not show up in the bar.

// SOCIAL MEDIA ICONS
var socialMediaIcons = {
  "twitter" : {
    "url" : "http://twitter.com/twitch",
    "icon" : "fa-twitter",
    "id" : "#twitterIcon",
  },
  "shop" : {
    "url" : "test",
    "icon" : "fa-shopping-cart",
    "id" : "#shopIcon"
  },

Twitter Feed

The only thing you really need to edit here is the "screenName" value, set it to the screen name of the Twitter account who's tweets you'd like to be displayed on the page.

// TWITTER CONFIG
var configList = {
    "profile": {
      "screenName": 'twitter'  //Your twitter username
    },
    "domId": 'tweetlist',
    "maxTweets": 5, //Number of tweets you want to show at a time
    "enableLinks": true,
    "showUser": false,
    "showTime": true,
    "showImages": false,
    "lang": 'en'
};

Setting up giveaway

To change the giveaway, you must have an account on Gleam.io

Then create a competition and choose the option to have a hosted landing page, this should give you a link like the one in the settings

Then simply change the link in the settings and it will be updated to your giveaway.

To get the "noGiveAwayMessage" text to show, delete the entry for the giveawayUrl, so it looks like "giveawayUrl" : ""

"giveAwayMessage" : "We are doing a giveaway..", //Message to show when the goal has been reached
"noGiveAwayMessage" : "There is no giveaway atm, sorry :(",
"giveawayUrl" : "https://gleam.io/YLJoq/my-giveaway" //Gleam.io giveaway URL, refer to documentation to see how to get this
